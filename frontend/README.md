# Lead Management System — Frontend

React SPA for the Lead Management System. See the [root README](../README.md) for full project documentation.

## Development

```bash
npm install
cp .env.example .env   # Configure API URL
npm run dev             # http://localhost:5173
```

## Build

```bash
npm run build           # Production SPA
npm run build:embed     # Embeddable chat widget
npm run lint            # Oxlint
```
